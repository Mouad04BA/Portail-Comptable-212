Understanding the Moroccan Chart of Accounts

Structure of Account Classes

The Moroccan Chart of Accounts is organized into 7 main classes:

  • Class 1: Financing accounts (equity, loans)
  • Class 2: Fixed assets accounts
  • Class 3: Current assets accounts (excluding cash)
  • Class 4: Current liabilities accounts (excluding cash)
  • Class 5: Cash accounts
  • Class 6: Expense accounts
  • Class 7: Revenue accounts

Each class is divided into accounts and sub-accounts with specific numbering conventions.

Coding System

The Moroccan accounting code uses a decimal system where:

  • The first digit indicates the class (1-7)
  • The first two digits indicate the account
  • The first three digits indicate the sub-account
  • The additional digits provide further detailed breakdowns

For example, in code 5116:

  • 5: Class 5 (Cash accounts)
  • 51: Treasury - Assets
  • 511: Checks and values to be cashed
  • 5116: Specific type of check/value
